# About project
*#unit tests in progress...* 

Imitation of VOD App inspired by frontendmentor.io challenge, link below. 

### Users are able to: 
- search media from search bar,
- select categories,
- register and login,
- save bookmarks (only logged in users)

### Used technologies: 
- React, Typescript,
- Redux,
- Router,
- Firebase,
- React Hook Form,
- React Lazy Load Image Component,
- React Burger Menu,
- Framer Motion,
- Mantine library,
- SASS
